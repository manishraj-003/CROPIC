# PROJECT REQUIREMENT DOCUMENT (PRD) CROPIC-AI for PMFBY

## Project Overview

CROPIC-AI is an AI-driven digital crop monitoring and insurance assessment platform under PMFBY. It enables real-time geo-tagged crop image capture, automated damage assessment using AI, and satellite-based cross-verification for faster and transparent insurance claim settlement.

## Objectives

1\. Enable real-time crop image capture.  
2. Automate crop damage assessment.  
3. Integrate NDVI-based satellite verification.  
4. Reduce claim settlement time.  
5. Improve transparency and farmer trust.

## Stakeholders

Farmers, Field Level Workers (FLWs), Insurance Companies, State Agriculture Departments, PMFBY Administrators.

## Functional Requirements

Mobile App: OTP login, geo-tagged image capture, offline support, multilingual voice guidance.  
  
AI Platform: Image quality validation, crop classification, damage classification, semantic segmentation for damage percentage, NDVI cross-verification.  
  
Web Dashboard: Real-time monitoring, map visualization, claim scoring, fraud alerts.

## Non-Functional Requirements

Scalable to 10M+ farmers, inference latency \< 3 seconds, AES-256 encryption, 99.5% uptime, GDPR/DPDP compliance.

## Success Metrics

Model accuracy ≥ 85%, segmentation IoU ≥ 80%, claim processing \< 7 days, reduction in disputes by 30% or more.
